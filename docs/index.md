# SMERVisual

SMERVisual is a Python package for explainable image classification with text-based explanations generated by large language models and analyzed with the Self Model Entities Related (SMER) method.

The package supports two main modes:

- OpenAI-hosted models for image descriptions and embeddings
- Local models for image descriptions, embeddings, and zero-shot object detection

## What You Can Do

- Generate short descriptions for images stored in class-specific folders
- Convert those descriptions into per-word embeddings
- Train a logistic regression model and compute SMER word importance
- Compare SMER and LIME with AOPC curves
- Draw bounding boxes for important words on images

## Typical Workflow

1. Generate descriptions with `image_descriptions()`
2. Create embeddings with `embed_descriptions()`
3. Aggregate embeddings and train a classifier
4. Run `classify_lr()` to compute importance scores
5. Visualize important words or AOPC curves
6. Optionally save annotated images with bounding boxes

## Package Layout

- `smer_visual.smer`: main public workflow functions
- `smer_visual.utils`: helper utilities used internally
